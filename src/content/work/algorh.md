## Application de planification intelligente

Solution de planification automatisée pour Centres de Relations Clients, générant des plannings optimisés en tenant compte des contraintes légales, règles métiers et préférences des collaborateurs.

![Happyplan](/assets/algorh.jpeg)

### Technologies utilisées
Laravel, MySQL, AWS Kinesis, OptaPlanner, SOAP, API Silae, API REST

![Happyplan](/assets/happyplan2.jpg)

### Réalisations
- Architecture et développement de l'API Laravel
- Conception de la base de données multi-clients pour la gestion des planifications
- Intégration d'AWS Kinesis pour le traitement en temps réel de l'activité des conseillers
- Connexion avec le système de planification externe OptaPlanner
- Intégration SOAP avec l'API Silae pour la synchronisation des données RH
- Développement des fonctionnalités métier (gestion des contraintes, préférences employés, courbes de charge)

### Impact
Réduction drastique du temps de planification pour les managers (de plusieurs jours à quelques heures). Solution personnalisable et éthique améliorant le bien-être des équipes tout en optimisant le staffing selon les besoins réels.
