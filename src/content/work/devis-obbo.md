## Application de génération de devis pour carrossiers

Solution web innovante permettant aux clients de générer automatiquement des devis de réparation à partir de photos de dommages sur leurs véhicules.

### Technologies utilisées
Laravel, FastAPI, MySQL, IA (classification d'images)

### Réalisations
- Conception et développement du parcours utilisateur complet
- Architecture de la base de données
- Développement de l'API Laravel pour la gestion des devis
- Création d'une API FastAPI intégrant un modèle de classification d'images par IA
- Refonte de l'identité graphique et du site web

### Impact
Application simplifiant radicalement le processus de demande de devis pour les clients tout en optimisant l'acquisition de prospects pour les carrossiers. Structure évolutive permettant l'amélioration continue du modèle d'IA.
