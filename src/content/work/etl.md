## Plateforme d'analyse pour chercheurs

Solution logicielle permettant l'extraction, le traitement et la visualisation automatisée des données expérimentales issues de fichiers variés (CSV, PDF, Excel).

### Technologies utilisées
Laravel, FastAPI, AWS Lambda, AWS Step Functions, Power BI, MySQL

### Réalisations
- Architecture et développement de l'API Laravel
- Conception de la base de données pour le stockage des résultats expérimentaux
- Mise en place de workflows ETL automatisés avec AWS Lambda et Step Functions
- Développement du système de pré-remplissage intelligent des données d'analyse
- Configuration de tableaux de bord Power BI pour la visualisation dynamique
- Implémentation d'imports par lots (batch processing)

### Impact
Automatisation complète de l'analyse des fichiers expérimentaux, réduisant drastiquement le temps de traitement manuel et permettant aux chercheurs de se concentrer sur l'interprétation des résultats via des visualisations Power BI personnalisables.
