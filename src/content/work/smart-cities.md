## Solutions Smart-Cities : iStoryPath et City2Gether

Développement de deux applications innovantes pour la ville intelligente : un guide touristique interactif et une plateforme citoyenne collaborative. Mes premiers projets professionnels en début de carrière.

### Technologies utilisées
Laravel, React, MySQL, API REST, Systèmes d'Information Géographique (SIG), NFC/QR Code

### iStoryPath - Guide touristique interactif

Application web permettant aux touristes de créer des itinéraires personnalisés selon leurs préférences, contraintes de temps et mode de transport. Récupération des parcours via NFC ou QR code, sans installation mobile nécessaire.

![iStoryPath](/assets/isp.jpg)

### City2Gether - Plateforme citoyenne

Application mobile favorisant les échanges entre habitants et collectivités via sondages, déclarations d'incidents et accès simplifié aux services municipaux.

![City2Gether](/assets/c2g.jpeg)

### Réalisations
- Architecture et développement des API Laravel pour les deux applications
- Conception des bases de données
- Intégration de données géographiques (SIG)
- Développement front-end avec React
- Implémentation des fonctionnalités NFC et QR code

### Impact
Amélioration de l'attractivité touristique des villes et renforcement de la citoyenneté collaborative, contribuant à une gestion urbaine plus connectée et participative.
